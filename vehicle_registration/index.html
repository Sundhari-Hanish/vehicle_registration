Department of Transport: Vehicle Management System

This centralized platform allows authorized users to manage the regional vehicle and owner registry. Use the navigation menu above to access specific database functions.

Available Services

  • People Search: Query the registry for owner details using names or driving license numbers.
  • Vehicle Search: Retrieve specific vehicle records including make, model, and ownership status.
  • Data Entry: Register new vehicles and update owner information within the secure Supabase environment.

General Information

Use this system to manage the transport database. All required fields must be filled for accurate data entry.
